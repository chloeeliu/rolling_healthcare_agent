### APACHE III Scoring Ranges
- **Total Score**: 0-299 points
- **Acute Physiology Score (APS)**: 0-252 points (17 variables)
- **Age Points**: 0-24 points
- **Chronic Health Evaluation**: 0-23 points

### Scoring Rules
- All acute physiology variables use **worst (most abnormal) value** in first 24 hours of ICU admission
- GCS uses **lowest** value in first 24 hours
- If FiO₂ < 0.5: use PaO₂ for oxygenation scoring
- If FiO₂ ≥ 0.5: use A-aDO₂ for oxygenation scoring
- A-aDO₂ = (FiO₂ × 713) − (PaCO₂ / 0.8) − PaO₂
- Creatinine points doubled if acute renal failure (ARF) present

```python
def first_day_vitalsign(stay_id):
    """
    Extract vital sign information for the first day (24 hours) of an ICU stay.

    This function retrieves heart rate, mean arterial pressure (MAP), and 
    respiratory rate measurements from the chartevents table for the period 
    from ICU admission (intime) to 24 hours after ICU admission.

    Parameters
    ----------
    stay_id : int
        The ICU stay identifier.

    Returns
    -------
    dict
        A dictionary containing first-day vital sign information:
        - 'stay_id': The ICU stay ID (int)
        - 'subject_id': Patient identifier (int)
        - 'hadm_id': Hospital admission ID (int)
        - 'intime': ICU admission time (str)
        - 'first_day_end': End of first day (intime + 24 hours) (str)
        - 'heart_rate': dict with 'min', 'max', 'avg', 'count'
        - 'map': dict with 'min', 'max', 'avg', 'count' (mean arterial pressure)
        - 'respiratory_rate': dict with 'min', 'max', 'avg', 'count'
        - 'has_hypotension': Boolean - True if minimum MAP < 65 mmHg
        - 'has_tachycardia': Boolean - True if maximum heart rate > 100 bpm
        - 'has_tachypnea': Boolean - True if maximum respiratory rate > 22 breaths/min
        - 'has_bradytachycardia': Boolean - True if heart rate < 60 or > 100 bpm

    Raises
    ------
    ValueError
        If stay_id is not provided or not found.
    """
```

```python
def first_day_lab(stay_id, subject_id, hadm_id):
    """
    Extract laboratory values for the first day (24 hours) of an ICU stay.

    This function retrieves key laboratory measurements from the ICU chart events
    during the first 24 hours after ICU admission. It provides summary statistics
    and clinical flags for common lab abnormalities.

    Parameters
    ----------
    stay_id : int, optional
        The ICU stay identifier. If provided, data is filtered to this specific stay.
    subject_id : int, optional
        The patient identifier. Can be used alone or with hadm_id.
    hadm_id : int, optional
        The hospital admission identifier. Used with subject_id if stay_id not provided.

    Returns
    -------
    dict
        A dictionary containing the following keys:
        - 'subject_id': patient identifier (int or None)
        - 'hadm_id': hospital admission identifier (int or None)
        - 'stay_id': ICU stay identifier (int or None)
        - 'intime': ICU admission time (str or None)
        - 'wbc_count': dict with WBC count statistics (K/uL or x10^9/L)
            - 'values': list of all WBC values
            - 'max': Maximum WBC value (float or None)
            - 'min': Minimum WBC value (float or None)
            - 'mean': Mean WBC value (float or None)
            - 'count': Number of WBC measurements (int)
            - 'has_elevated_wbc': Boolean indicating if any WBC > 12 K/uL
        - 'platelet_count': dict with platelet count statistics (K/uL)
            - 'values': list of all platelet values
            - 'max': Maximum platelet value (float or None)
            - 'min': Minimum platelet value (float or None)
            - 'mean': Mean platelet value (float or None)
            - 'count': Number of platelet measurements (int)
            - 'has_thrombocytopenia': Boolean indicating if any platelet < 150 K/uL
        - 'creatinine': dict with creatinine statistics (mg/dL)
            - 'values': list of all creatinine values
            - 'max': Maximum creatinine value (float or None)
            - 'min': Minimum creatinine value (float or None)
            - 'mean': Mean creatinine value (float or None)
            - 'count': Number of creatinine measurements (int)
        - 'all_values': DataFrame with all lab values during the first day
            Columns: charttime, valuenum, parameter, unit

    Raises
    ------
    ValueError
        If no patient identifier is provided.

    Notes
    -----
    This function uses the following item IDs from the MIMIC-IV database:
    - WBC: 220546 (K/uL)
    - Platelet Count: 227457 (K/uL)
    - Creatinine (serum): 220615 (mg/dL)
    - Creatinine (whole blood): 229761 (mg/dL)

    Clinical thresholds used:
    - Elevated WBC: > 12 K/uL (x10^9/L)
    - Thrombocytopenia: platelet count < 150 K/uL
    """
```

```python
def vitalsign(stay_id):
    """
    Extract vital sign information for a patient's ICU stay.

    This function queries the MIMIC-IV database to retrieve vital sign measurements
    (heart rate, blood pressure) for a specific ICU stay and computes summary statistics
    and clinical flags.

    Parameters:
    -----------
    stay_id : int
        The ICU stay identifier (stay_id from mimiciv_icu.icustays table)

    Returns:
    --------
    dict : A dictionary containing:
        - 'stay_id': The input stay_id
        - 'heart_rate': dict with min, max, mean, count of heart rate measurements (bpm)
        - 'has_tachycardia': bool - True if any heart rate > 100 bpm
        - 'systolic_bp': dict with min, max, mean, count of systolic BP measurements (mmHg)
        - 'diastolic_bp': dict with min, max, mean, count of diastolic BP measurements (mmHg)
        - 'mean_arterial_pressure': dict with min, max, mean, count of MAP measurements (mmHg)
        - 'has_hypotension': bool - True if any MAP < 65 mmHg
        - 'vital_signs_data': DataFrame with all raw vital sign measurements

    Clinical Definitions:
    --------------------
    - Hypotension: Mean arterial pressure (MAP) < 65 mmHg at any point during the stay
    - Tachycardia: Heart rate > 100 bpm at any point during the stay

    Vital Sign Item IDs Used:
    -------------------------
    - Heart rate: 220045 (bpm)
    - Arterial BP systolic: 220050 (mmHg)
    - Non-invasive BP systolic: 220179 (mmHg)
    - Arterial BP diastolic: 220051 (mmHg)
    - Non-invasive BP diastolic: 220180 (mmHg)
    - Arterial BP mean: 220052 (mmHg)
    - Non-invasive BP mean: 220181 (mmHg)

    Notes:
    ------
    - If no measurements exist for a vital sign type, the stats dict will have
      None values for min, max, mean and count of 0
    - The function combines both arterial and non-invasive blood pressure measurements
    """
```

### APACHE III Scoring Ranges
- **Total Score**: 0-299 points
- **Acute Physiology Score (APS)**: 0-252 points (17 variables)
- **Age Points**: 0-24 points
- **Chronic Health Evaluation**: 0-23 points

### Scoring Rules
- All acute physiology variables use **worst (most abnormal) value** in first 24 hours of ICU admission
- GCS uses **lowest** value in first 24 hours
- If FiO₂ < 0.5: use PaO₂ for oxygenation scoring
- If FiO₂ ≥ 0.5: use A-aDO₂ for oxygenation scoring
- A-aDO₂ = (FiO₂ × 713) − (PaCO₂ / 0.8) − PaO₂
- Creatinine points doubled if acute renal failure (ARF) present
- **ICD codes in MIMIC stored without decimal points** (e.g., 571.5 → 5715)

### APACHE III Scoring Ranges
- **Total Score**: 0-299 points
- **Acute Physiology Score (APS)**: 0-252 points (17 variables)
- **Age Points**: 0-24 points
- **Chronic Health Evaluation**: 0-23 points

### Scoring Rules
- All acute physiology variables use **worst (most abnormal) value** in first 24 hours of ICU admission
- GCS uses **lowest** value in first 24 hours
- If FiO₂ < 0.5: use PaO₂ for oxygenation scoring
- If FiO₂ ≥ 0.5: use A-aDO₂ for oxygenation scoring
- A-aDO₂ = (FiO₂ × 713) − (PaCO₂ / 0.8) − PaO₂
- Creatinine points doubled if acute renal failure (ARF) present
- **ICD codes in MIMIC stored without decimal points** (e.g., 571.5 → 5715)

### Admission Type Classification (MIMIC-IV)
| Admission Type | Classification |
|----------------|----------------|
| ELECTIVE | Elective postoperative |
| SURGICAL SAME DAY ADMISSION | Emergency postoperative |
| URGENT (not OBSERVATION) | Emergency postoperative |
| EW EMER. | Emergency non-operative |
| DIRECT EMER. | Emergency non-operative |
| OBSERVATION types | Non-operative |

### APACHE III Scoring Ranges
- **Total Score**: 0-299 points
- **Acute Physiology Score (APS)**: 0-252 points (17 variables)
- **Age Points**: 0-24 points
- **Chronic Health Evaluation**: 0-23 points

### Scoring Rules
- All acute physiology variables use **worst (most abnormal) value** in first 24 hours of ICU admission
- GCS uses **lowest** value in first 24 hours
- If FiO₂ < 0.5: use PaO₂ for oxygenation scoring
- If FiO₂ ≥ 0.5: use A-aDO₂ for oxygenation scoring
- A-aDO₂ = (FiO₂ × 713) − (PaCO₂ / 0.8) − PaO₂
- Creatinine points doubled if acute renal failure (ARF) present
- **ICD codes in MIMIC stored without decimal points** (e.g., 571.5 → 5715)
- **Missing values**: If a lab test is not measured, assign 0 points (assume normal)
- **Albumin and bilirubin**: Optional, default to 0 points if not measured
- **GCS under sedation**: If no pre-sedation GCS available, use GCS=15 (normal) = 0 points

### Chronic Health Evaluation Rules (CRITICAL UPDATE)
- **Only the SINGLE HIGHEST applicable chronic health score is used (NOT additive)**
- Scoring depends on admission type:

| Condition | Elective Postoperative | Emergency Postoperative or Non-operative |
|-----------|----------------------|-----------------------------------------|
| AIDS | 23 | 11 |
| Hepatic failure | 4 | 4 |
| Lymphoma | 13 | 13 |
| Metastatic cancer | 11 | 14 |
| Leukemia / myeloma | 10 | 10 |
| Immunosuppression | 10 | 10 |

### Chronic Conditions Definitions
- **AIDS**: Confirmed HIV with opportunistic infection or Kaposi sarcoma
- **Hepatic failure**: Biopsy-proven cirrhosis, portal hypertension, prior hepatic encephalopathy or variceal bleeding
- **Lymphoma**: Documented lymphoma diagnosis
- **Metastatic cancer**: Documented distant metastases
- **Leukemia / multiple myeloma**: Documented hematologic malignancy
- **Immunosuppression**: Immunosuppressive therapy, chemotherapy, radiation, high-dose steroids, or disease-related immunocompromise
- **Chronic renal failure**: Chronic dialysis dependency (ESRD) - handled separately

### Admission Type Classification (MIMIC-IV)
| Admission Type | Classification |
|----------------|----------------|
| ELECTIVE | Elective postoperative |
| SURGICAL SAME DAY ADMISSION | Emergency postoperative |
| URGENT (not OBSERVATION) | Emergency postoperative |
| EW EMER. | Emergency non-operative |
| DIRECT EMER. | Emergency non-operative |
| OBSERVATION types | Non-operative |

    Returns

        - 'subject_id': Patient identifier
        - 'hadm_id': Hospital admission ID
        - 'intime': ICU admission time
        - 'aps_score': Acute Physiology Score (0-252)
        - 'age_points': Age points (0-24)
        - 'chronic_health_points': Chronic health points (0-23)
        - 'total_score': Total APACHE III score (0-299)
        - 'component_scores': Dictionary with individual component scores

    SELECT subject_id, hadm_id, intime
    FROM mimiciv_icu.icustays
    WHERE stay_id = ?

    hadm_id = int(stay_info.iloc[0]['hadm_id'])
    intime = stay_info.iloc[0]['intime']
    
    # Calculate first day end time (24 hours from ICU admission)
    intime_dt = datetime.strptime(str(intime), '%Y-%m-%d %H:%M:%S')
    first_day_end = intime_dt + timedelta(hours=24)
    first_day_end_str = first_day_end.strftime('%Y-%m-%d %H:%M:%S')
    
    # Get patient age
    sql = """
    SELECT anchor_age
    FROM mimiciv_hosp.patients
    WHERE subject_id = ?
    """
    patient_info = query_db(sql, params=[subject_id])
    age = int(patient_info.iloc[0]['anchor_age']) if len(patient_info) > 0 else None
    
    # Get admission type for chronic health evaluation
    sql = """
    SELECT admission_type
    FROM mimiciv_hosp.admissions
    WHERE hadm_id = ?
    """
    admission_info = query_db(sql, params=[hadm_id])
    admission_type = admission_info.iloc[0]['admission_type'] if len(admission_info) > 0 else None
    
    # Determine if postoperative (elective vs emergency)
    # Based on MIMIC-IV admission types:
    # - ELECTIVE = elective postoperative
    # - SURGICAL SAME DAY ADMISSION = emergency postoperative
    # - URGENT (not OBSERVATION) = emergency postoperative
    # - EW EMER., DIRECT EMER. = emergency non-operative
    # - OBSERVATION types = non-operative
    admission_type_str = str(admission_type).upper() if admission_type else ""
    
    is_elective_postop = 'ELECTIVE' in admission_type_str
    is_emergency_postop = 'SURGICAL SAME DAY' in admission_type_str or ('URGENT' in admission_type_str and 'OBSERVATION' not in admission_type_str)
    is_non_operative = not is_elective_postop and not is_emergency_postop
    
    # Get all chartevents for first 24 hours
    sql = """
    SELECT itemid, valuenum, charttime
    FROM mimiciv_icu.chartevents
    WHERE stay_id = ?
      AND charttime >= ?
      AND charttime <= ?
      AND valuenum IS NOT NULL
    """
    chartevents = query_db(sql, params=[stay_id, intime, first_day_end_str])
    
    # Create lookup for chartevents by itemid
    def get_values(itemids):
        """Get all values for given itemids"""
        if isinstance(itemids, int):
            itemids = [itemids]
        df = chartevents[chartevents['itemid'].isin(itemids)]
        return df

    hr_df = get_values(220045)  # Heart Rate
    map_df = get_values([220052, 220181])  # MAP (arterial and non-invasive)
    rr_df = get_values(220210)  # Respiratory Rate
    temp_df = get_values(223762)  # Temperature Celsius
    
    # Get lab values
    hct_df = get_values(220545)  # Hematocrit
    wbc_df = get_values(220546)  # WBC
    creat_df = get_values(220615)  # Creatinine
    bun_df = get_values(225624)  # BUN
    na_df = get_values(220645)  # Sodium
    glucose_df = get_values(220621)  # Glucose
    albumin_df = get_values(227456)  # Albumin
    bili_df = get_values(225690)  # Bilirubin
    
    # Get ABG values
    ph_df = get_values(223830)  # pH
    paco2_df = get_values(220235)  # PaCO2
    pao2_df = get_values(220224)  # PaO2

### APACHE III Scoring Ranges
- **Total Score**: 0-299 points
- **Acute Physiology Score (APS)**: 0-252 points (17 variables)
- **Age Points**: 0-24 points
- **Chronic Health Evaluation**: 0-23 points

### Scoring Rules
- All acute physiology variables use **worst (most abnormal) value** in first 24 hours of ICU admission
- GCS uses **lowest** value in first 24 hours
- If FiO₂ < 0.5: use PaO₂ for oxygenation scoring
- If FiO₂ ≥ 0.5: use A-aDO₂ for oxygenation scoring
- A-aDO₂ = (FiO₂ × 713) − (PaCO₂ / 0.8) − PaO₂
- Creatinine points doubled if acute renal failure (ARF) present
- **ICD codes in MIMIC stored without decimal points** (e.g., 571.5 → 5715)
- **Missing values**: If a lab test is not measured, assign 0 points (assume normal)
- **Albumin and bilirubin**: Optional, default to 0 points if not measured
- **GCS under sedation**: If no pre-sedation GCS available, use GCS=15 (normal) = 0 points

### Chronic Health Evaluation Rules (CRITICAL)
- **Only the SINGLE HIGHEST applicable chronic health score is used (NOT additive)**
- Scoring depends on admission type:

| Condition | Elective Postoperative | Emergency Postoperative or Non-operative |
|-----------|----------------------|-----------------------------------------|
| AIDS | 23 | 11 |
| Hepatic failure | 4 | 4 |
| Lymphoma | 13 | 13 |
| Metastatic cancer | 11 | 14 |
| Leukemia / myeloma | 10 | 10 |
| Immunosuppression | 10 | 10 |

### Chronic Conditions Definitions
- **AIDS**: Confirmed HIV with opportunistic infection or Kaposi sarcoma (ICD: 042, 043.x, V08)
- **Hepatic failure**: Biopsy-proven cirrhosis, portal hypertension, prior hepatic encephalopathy or variceal bleeding (ICD: 571.2-571.9, 456.0-456.2, 572.2-572.4, 573.2-573.3)
- **Lymphoma**: Documented lymphoma diagnosis (ICD: 200.x, 202.x, 203.x)
- **Metastatic cancer**: Documented distant metastases (ICD: 196.x, 197.x, 198.x, 199.x)
- **Leukemia / multiple myeloma**: Documented hematologic malignancy (ICD: 204.x-208.x, 203.0-203.9)
- **Immunosuppression**: Immunosuppressive therapy, chemotherapy, radiation, high-dose steroids, or disease-related immunocompromise (ICD: V58.1, V58.61-63, V58.67)
