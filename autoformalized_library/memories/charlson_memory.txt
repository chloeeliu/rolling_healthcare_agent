## Error Fixes

1. **ICD-9 Code Format**: ICD-9 codes stored WITHOUT decimal points (e.g., '5715' not '571.5'). Updated all ICD-9 code lists accordingly.

2. **ICD-10 Code Format**: ICD-10 codes stored WITHOUT decimal points (e.g., 'E1122' not 'E11.22', 'I252' not 'I25.2'). Updated all ICD-10 code lists to remove decimal points.

3. **Duplicate Condition Counting**: Changed from adding points in loop to using sets to track unique conditions, then calculating score after loop.

4. **Condition Weights**: Fixed hemiplegia, renal disease, leukemia, lymphoma to be 2 points (not 1). Used separate sets for 1pt, 2pt, 3pt, 6pt conditions.

5. **numpy.int64 Type Error**: When accessing DataFrame columns, convert to Python int using `int(df.iloc[0]['subject_id'])`.

6. **Pandas Column Access**: Use `df['column_name'].iloc[0]` instead of `df.iloc[0]['column_name']` for proper column access.

7. **Missing ICD-10 Codes**: Added missing codes:
   - I252 (old myocardial infarction) to MI codes
   - J81 (pulmonary edema) to CHF codes
   - I255 (ischemic cardiomyopathy) to CHF codes
   - N17 (acute kidney failure) to renal codes
   - Updated diabetes complication codes to match database format (without decimals)

8. **Diabetes Complication Code Format**: Changed from 'E11.2' to 'E112' format to match database storage.

```python
    for condition in found_6pt:
        conditions[condition] = 6
        cci_score += 6
    
    if has_diabetes_comp:
        conditions['diabetes_with_end_organ_damage'] = 2
        cci_score += 2
    elif has_diabetes_no_comp:
        conditions['diabetes_without_end_organ_damage'] = 1
        cci_score += 1
    
    if has_severe_liver:
        conditions['moderate_severe_liver_disease'] = 3
        cci_score += 3
    elif has_mild_liver:
        conditions['mild_liver_disease'] = 1
        cci_score += 1
    
    if has_metastatic:
        conditions['metastatic_solid_tumor'] = 6
        cci_score += 6
    elif has_any_tumor:
        conditions['any_tumor'] = 2
        cci_score += 2
    
    return {
        'cci_score': cci_score,
        'conditions': conditions,
        'has_cohort': cci_score >= 3
    }
```
